# Variables, if, else, while


## Description
This directory contains C programs that practice the use of variables, conditional statements (if, else) and while loops, as part of the Holberton School low-level programming curriculum.

## Author
**Sagalou** - Holberton School Student

## Files

| File | Description |
|------|-------------|
| `0-positive_or_negative.c` | Program that determines if a random number is positive, negative, or zero |

## Requirements
- Ubuntu 20.04 LTS
- gcc with flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Betty coding style compliant

## Compilation
```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 <filename>.c -o <output>
```
